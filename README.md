\# Smart Travel Planner



Smart Travel Planner is a Spring Boot based web application developed to help users plan trips efficiently.



\## Tech Stack

\- Java

\- Spring Boot

\- Maven

\- MySQL

\- HTML

\- CSS

\- Eclipse IDE



\## About

This project is built using Spring Boot.

Maven is used to manage dependencies and build the project.

The application was developed and run using Eclipse IDE.



\## Build \& Run Steps



\### Prerequisites

\- Java JDK 17

\- Maven

\- MySQL

\- Eclipse IDE

\- Git



\### Steps to Run (Using Eclipse)



1\. Clone the repository  

&nbsp;  git clone https://github.com/Navya-S-Shetty/Smart-Travel-Planner.git



2\. Open Eclipse IDE



3\. Import the project  

&nbsp;  File → Import → Existing Maven Project



4\. Configure database in  

&nbsp;  src/main/resources/application.properties



5\. Build the project  

&nbsp;  Right click project → Run As → Maven Build (clean install)



6\. Run the application  

&nbsp;  Right click TripplannerApplication.java → Run As → Spring Boot App



7\. Open browser  

&nbsp;  http://localhost:8080



\## Notes

\- Maven is used only for build and dependency management.

\- Spring Boot is used to run the application.

\- Database credentials and API keys should be configured locally.
